Project Write-Up

Rationale: This visualization explores how heart rate and respiratory (breathing) rate respond under physical (Running) and cognitive (2‑Back) load compared to Rest. We chose a scatter plot so that each data point is represented individually; by default they appear in gray to avoid visual clutter. Interaction via legends (checkboxes) enables users to “highlight” data points for each activity and selectively view heart rate or breathing rate.

Development Process: The project was developed using HTML, CSS, and JavaScript with D3.js as the only plotting library. The work was divided between data processing, layout design, and interactivity, and took roughly 30 person‑hours to complete.
